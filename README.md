# Sistema de Gestión de Pedidos

API REST desarrollada con Spring Boot para administrar clientes, categorías, productos y pedidos.

## Tecnologías
- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- AOP
- Validation
- Postman
